# 🚀 Bidhan Email Generator Pro

**Bidhan Email Generator Pro** is a powerful, high-speed, and secure web-based tool designed to generate high-quality random emails instantly. Developed with a modern "Fire" themed UI, it supports multiple global domains and localized name formats.

---

## 🔥 Key Features

- **Global Name Support:** Generate realistic emails using names from **Bangladesh**, **USA**, and **India**.
- **Multiple Domain Options:** Choose from popular providers like **Gmail, Yahoo, Outlook, Hotmail, iCloud, AOL, Zoho**, or use a **Random Mix**.
- **Smart Copy System:** Copy individual emails or use the **"Copy All"** feature for bulk actions with a single click.
- **Premium UI/UX:** Features a professional dark-themed interface with smooth fire animations and interactive elements.
- **Vibration Feedback:** Haptic feedback on mobile devices for a better user experience.
- **Lightweight & Fast:** Built with pure HTML, CSS, and JavaScript for maximum performance.

---

## 🛠️ How to Use

1. **Select Quantity:** Choose how many emails you want to generate (up to 100 at a time).
2. **Select Country:** Pick a country to set the name format for the emails.
3. **Choose Domain:** Select your preferred email provider or a random mix.
4. **Generate:** Click the **"Generate Emails"** button.
5. **Copy:** Click the **"Copy"** button next to any email, or use **"Copy All Emails"** to save the entire list.
